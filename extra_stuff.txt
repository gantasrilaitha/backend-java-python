Marketing teams invest significant time in planning campaigns, creating creatives, deciding budgets, and analyzing competitors. Prarochana AI automates these activities using Generative AI, enabling faster, smarter, and data-driven campaign creation.

b. Key Features / Components
AI-Powered Campaign Generation
Automated ad caption generation using GPT-based models
AI-generated advertisement creatives using text-to-image models
Intelligent campaign budget estimation using RAG
Competitor-aware campaign recommendations
End-to-end campaign generation pipeline
Technical Components
Fine-tuned GPT2 & Gemini models
DALL·E/BingArt for image generation
RAG for budget prediction
Pinecone & MongoDB Vector Search
Streamlit-based interactive application
